# Notebooks

These notebooks are placeholders for the research workflow:

- `dataset_exploration.ipynb`: explore dataset distribution, label mapping, class imbalance
- `model_training.ipynb`: experiment with transfer learning (ResNet50 / EfficientNet)
- `evaluation.ipynb`: metrics (accuracy, precision, recall, F1, confusion matrix)
- `grad_cam_explainability.ipynb`: Grad-CAM visualizations for research interpretability

Tip: keep the **backend preprocessing** (`backend/app/services/preprocessing.py`) aligned with notebook preprocessing to avoid train/serve skew.
